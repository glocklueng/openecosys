This is a 4-layers PCB with 2x silkscreen and 2x soldermask.
1 Oz copper on all layers.
Black soldermask and white silkscreen.

Top Layer 		(.GTL)
Inner plane 	(.GP1)
Inner layer		(.G1)
Bottom Layer	(.GBL)

Extensions:
===========

.GTL	Gerber Top Layer (component side)
.GP1	Gerber Inner Layer (internal ground plane) [Negative]
.G1		Gerber Internal Layer (internal routing layer/plane) [Positive]
.GBL	Gerber Bottom Layer (solder side)
.GTO	Gerber Top Overlay (top silkscreen)
.GBO	Gerber Bottom Overlay (bottom silkscreen)
.GTS	Gerver Top Soldermask
.GBS	Gerver Bottom Soldermask
.GM4	Board outline
.TXT	Excellon drills
